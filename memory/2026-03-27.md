### Root Cause: False Error Status on Cron Jobs

- **Fix:** Hardcoded Discord channel IDs in `delivery.to` instead of env vars

- **Result:** After hardcoding, all jobs report `status: "ok"` and `deliveryStatus: "delivered"` correctly
